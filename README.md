# Smart-Weather-App
The Smart Weather App is a responsive frontend web application built using HTML, CSS, and JavaScript that fetches real-time weather data using a REST API. It allows users to search for any city and instantly view current weather conditions with a clean and modern UI.

This project demonstrates frontend development skills, API integration, asynchronous JavaScript, and UI/UX best practices.
